# ==================== 拦截规则语法 ====================
# ||domain.com^     - 拦截该域名及其所有子域名
# @@||domain.com^   - 白名单（见 whitelist.txt）
# |domain.com^      - 拦截精确匹配的域名
# ||example.com/ads/* - 拦截特定路径
# ||example.com^$third-party - 拦截第三方请求

# ==================== 添加方式 ====================
# 方法 1: Web 界面
#   1. 登录 AdGuard Home（默认端口 3000）
#   2. 过滤器 → DNS 黑名单 → 添加阻止列表
#   3. 输入列表 URL 和名称
#   4. 点击保存

# 方法 2: 配置文件（不推荐手动编辑）
#   编辑 /etc/AdGuardHome.yaml
#   在 filters 部分添加新列表
#   重启 AdGuard Home

# ==================== 注意事项 ====================
# 1. 过多的过滤列表会增加 DNS 查询延迟
# 2. 建议启用 5-8 个高质量列表即可
# 3. 自定义规则优先级高于上游列表
# 4. 使用白名单解决误拦截问题
# 5. AdGuard Home 会自动更新过滤列表（默认 24 小时）
# 6. 定期检查查询日志，优化过滤规则
